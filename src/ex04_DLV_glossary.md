#### Атрибуты новых стейкхолдеров

**st01000 - Система бухгалтерского учета:**
- Тип: Внешняя ИТ-система
- Взаимодействие: Получает данные о заказах и доставке автоматически, передает данные о начисленной оплате
- Критичность: Высокая (необходима для учета и расчетов)
- Формат данных: Структурированные данные (ID заказа, поставщик, курьер, сумма, дата)

**st01100 - Мобильное приложение курьера:**
- Тип: Техническая система (часть разрабатываемой системы)
- Взаимодействие: Основной интерфейс для курьеров
- Критичность: Критическая (основной инструмент курьера)
- Требования: Работа в офлайн-режиме, геолокация, простота использования

**st01200 - Веб-интерфейс диспетчера:**
- Тип: Техническая система (часть разрабатываемой системы)
- Взаимодействие: Интерфейс для диспетчера
- Критичность: Высокая (управление заказами и курьерами)
- Требования: Real-time обновления, карты, фильтры и сортировка

**st01300 - Рабочее место администратора:**
- Тип: Техническая система (часть разрабатываемой системы)
- Взаимодействие: Интерфейс для администратора
- Критичность: Высокая (управление пользователями и доступом)
- Требования: Безопасность, аудит действий, управление правами

**st01400 - Веб-интерфейс оператора:**
- Тип: Техническая система (часть разрабатываемой системы)
- Взаимодействие: Интерфейс для оператора
- Критичность: Высокая (ввод заказов в систему)
- Требования: Валидация данных, удобство ввода, обработка ошибок
