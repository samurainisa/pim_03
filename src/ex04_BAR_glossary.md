#### Атрибуты новых стейкхолдеров

**st00900 - Система бухгалтерского учета:**
- Тип: Внешняя ИТ-система
- Взаимодействие: Получает данные об оплате услуг автоматически
- Критичность: Высокая (необходима для учета)
- Формат данных: Структурированные данные (ID записи, клиент, услуга, сумма, дата)

**st01000 - Веб-интерфейс системы:**
- Тип: Техническая система (часть разрабатываемой системы)
- Взаимодействие: Интерфейс для всех пользователей (клиенты, посетители, менеджеры, мастера)
- Критичность: Критическая (основной способ взаимодействия)
- Требования: Доступность, удобство использования, безопасность

**st01100 - API систем уведомлений:**
- Тип: Внешние API (Telegram, WhatsApp, VK, СМС-провайдеры)
- Взаимодействие: Отправка уведомлений клиентам
- Критичность: Средняя (улучшает пользовательский опыт)
- Формат данных: Текст сообщения, контакт получателя, тип уведомления
