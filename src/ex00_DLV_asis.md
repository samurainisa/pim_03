### Проблемы, выявленные в текущем состоянии:

1. **Для курьеров:**
   - Нет доступа к информации о заказах
   - Нужно звонить диспетчеру для получения заданий
   - Нет механизма выбора и бронирования заказов
   - Нет системы для отражения выполненных действий
   - Нет информации об оплате

2. **Для диспетчера:**
   - Нет централизованного контроля курьеров
   - Сложно оперативно переназначать заказы
   - Нужно звонить курьерам для получения статуса
   - Нет системы отслеживания заказов

3. **Для оператора:**
   - Разные форматы заказов от разных поставщиков
   - Трудоемкая конвертация в единый формат
   - Возможны ошибки при ручном вводе
   - Нет системы для ввода заказов

4. **Для магазинов и предприятий питания:**
   - Нет единого формата передачи заказов
   - Нет возможности отслеживать статус доставки
   - Нет обратной связи о выполнении заказов

5. **Для клиентов:**
   - Нет информации о времени доставки
   - Непредсказуемость доставки
   - Нет возможности отслеживать заказ

6. **Для бухгалтерии:**
   - Нет автоматической передачи данных о заказах
   - Ручной расчет оплаты
   - Возможны ошибки при ручном вводе
   - Нет системы для передачи данных об оплате

7. **Для администратора:**
   - Нет системы управления пользователями
   - Нет системы управления доступом

8. **Для студентов-основателей:**
   - Нет централизованной системы учета
   - Невозможно отслеживать эффективность работы
